# 📊 Calculateur de Moyenne

Un petit programme en Python pour calculer sa moyenne de notes, fait dans le cadre de mes premiers projets en L1 Info.

## 📋 Description

Ce programme permet de :
- Entrer ses notes une par une (entre 0 et 20)
- Calculer automatiquement la moyenne
- Obtenir un petit commentaire selon le résultat

## 🚀 Lancer le programme

Aucune installation nécessaire, juste Python.

```bash
python calculateur_moyenne.py
```

## 💻 Exemple d'utilisation

```
========================================
     SALUT ! JE CALCULE TES MOYENNES
========================================

========================================
      CALCULATEUR DE MOYENNE
========================================
1. Calculer ma moyenne
2. Quitter
----------------------------------------
Tu veux faire quoi ? 1

----------------------------------------
Entrez vos notes une par une
Quand vous avez fini, appuyez sur Entrée
----------------------------------------

Note 1 : 14
Note 2 : 12
Note 3 : 16
Note 4 : 

========================================
            RESULTAT
========================================
Nombre de notes : 3
Tes notes : [14.0, 12.0, 16.0]
Moyenne : 14.00/20
Bien joué !
========================================
```
